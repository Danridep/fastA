Как работает анализ

Pandas анализ выполняет следующие действия:

  1. Загружает Excel файл, пропуская служебные строки
  2. Фильтрует данные по выбранному месяцу
  3. Группирует данные по филиалам ("Родитель")
  4. Суммирует количество и стоимость
  5. Добавляет общий итог
  6. Сохраняет результат в новый Excel файл
